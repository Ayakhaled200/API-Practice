# 🧪 API Testing Practice Collections

This repository contains practice API test collections created using Postman.
The goal is to experiment with request handling, response validation, and basic automation workflows.

---

## 📦 Collections Included

### 1️⃣ Reqres

Practice collection targeting the public Reqres demo API.

**Covers:**
- User listing and single user retrieval
- Create, update, and delete user
- Register and login scenarios
- Basic response time validation

**Required Variable:**
- `base_url`

---

### 2️⃣ Trello

Practice collection for testing Trello REST API endpoints.

**Covers:**
- Boards, Lists, Cards, and Checklists operations
- Basic CRUD flows
- Response time validation

**Required Variables:**
- `APIKey`
- `APIToken`
- Entity IDs (`boardID`, `listID`, etc.)
- `base_url`
